# KB-005: Software Requests & Access Changes

**Approved software list** (client-standard, no security review needed): Adobe Acrobat Pro, Microsoft Visio, Zoom, Slack, 7-Zip, Chrome, Firefox.

**Process for approved-list software:** user's **manager approval** required (email or portal sign-off) → install completed within **1 business day** of approval. Software NOT on the list requires a security review (tier 2, 3–5 business days).

**Access changes (shared mailboxes, distribution lists, drive permissions):** require **manager approval** before any change. Support must never state that access has already been granted or software already installed — describe the process and next step only.

**Multi-user line-of-business app failures (QuickBooks, practice management, dispatch systems):** never a self-service runbook — escalate tier 2 with scope, trigger, and business deadline in the handoff summary.
