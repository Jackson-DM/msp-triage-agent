# KB-007: Onboarding & Offboarding

**Onboarding:** provisioning workflow (accounts, hardware, licenses, system access) — always a tier-2 handoff. Standard lead time: **3 business days** before start date. Handoff summary must capture: name, role, start date, requested systems, hardware needs, and the reference employee if "same as X" is specified.

**Voluntary offboarding (resignation, planned departure):** scheduled for the final day; tier 2, medium priority.

**Involuntary termination:** access revocation is **same-day / immediate** — disable accounts, revoke remote access and sessions, collect hardware. Priority: **critical**, tier 2, and note in the handoff if the role had elevated access (security-team involvement).
